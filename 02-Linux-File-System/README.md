## What I Learned
The Linux file system starts from the root directory (/). Every directory has a specific purpose. Understanding the file system helps cloud engineers locate configuration files, logs, applications, and user data.
## Real World Usage
Cloud engineers frequently work with directories such as:

- `/etc` for configuration files
- `/var/log` for logs
- `/home` for user data
- `/tmp` for temporary files
